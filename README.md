# Live Present

Aplikasi presentasi interaktif berbasis Node.js yang memungkinkan audiens memberikan reaksi secara realtime saat presentasi berlangsung. File Presentasi berupa file PDF.

## ✨ Features

* ⚡ Realtime reaction system
* ❤️ Emoji reactions langsung tampil tanpa reload
* 📱 Responsive UI untuk HP & desktop
* 🔌 WebSocket realtime communication
* 🖥️ Presenter mode
* 🌙 Modern interface
* 🚀 Lightweight & fast

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* Socket.IO
* HTML/CSS/JavaScript

---

## 📦 Installation

Clone repository:

```bash
git clone https://github.com/ih24n69/live-present.git
```

Masuk ke folder project:

```bash
cd live-present
```

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Application

Menjalankan server:

```bash
node server.js
```

---

## 🌐 Access Application

Setelah server berjalan, buka:

```txt
http://localhost:9090
```

Ganti port sesuai konfigurasi aplikasi kamu.

---

## 🚀 Production Run

Menggunakan PM2:

```bash
npm install -g pm2
pm2 start server.js --name live-present
```
